# DashboardVendasExcel

Foi realizado um Dashboard no Excel a partir de uma base de dados contendo informações de vendas de jogos.
Nele, foi utilizado o método ABCDE, no qual consiste em destinar uma aba para cada uma das seguintes etapas:
A - Assets: onde são colocados todos os dispositivos necessários para realizar o dashboard (paleta de cores com os respectivos códigos, ícones, logos ou outras imagens necessárias etc);
B - Bases: Dados base para construção do dashboard;
C - Cálculos: Cálculos necessários (tabelas dinâmicas etc0;
D - Dashboard: Construção do dashboard utilizando os dados e materiais das planilhas anteriores.
